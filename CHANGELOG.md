## [0.1.3] - 2026-05-26

### Added
- idle 自动备份新增用户可配置开关和 6 / 12 / 24 / 48 小时间隔。
- 关于弹窗新增本地存储用量展示。
- 新增 IndexedDB quota 监控：接近容量上限时提示用户，达到临界值时拒绝创建新快照。
- 自动安全快照保留策略支持按容量限制，选项为 5 / 20 / 50 / 100 MB。
- 选择性导出新增按 tag 过滤日志，多个 tag 使用 OR 语义。

### Changed
- 数据快照设置区合并展示备份偏好、自动备份间隔、保留方式和保留值。
- 导出文件名在按 tag 筛选时追加 `tags-N` 后缀，避免与全量导出混淆。

### Deferred
- Service Worker periodicSync 自动备份因当前 PWA 构建仍为 generateSW 模式，推迟到后续 PWA 整顿。

## [0.1.2] - 2026-05-26

### Added
- 建立版本号单一来源，应用显示、导出元信息与构建版本统一来自 `package.json`。
- 新增关于弹窗，集中展示应用版本、数据版本、IndexedDB 版本、隐私说明与更新历史。
- CSV 导出包新增 `meta.json`，记录 `appVersion`、`dataVersion` 与导出时间。
- PWA 更新改为用户可见提示，用户点击刷新后才切换到新版本。
- 备份快照写入后会立即读回自检，发现异常会回滚失败快照并提示错误。
- 新增 idle 自动安全快照，应用空闲且距离上次自动快照超过 24 小时时自动备份。
- 导入合并新增字段级冲突预览，可选择全部保留当前值或全部使用导入值。
- 新增导出选项弹窗，支持按日期区间、数据维度和 JSON / CSV / Markdown 格式选择性导出。
- 自动安全快照保留数可配置为 3 / 7 / 15 / 30，手动快照继续不限数量。

### Changed
- 导入预览明确区分 `dataVersion` 与 `appVersion`：只有 `dataVersion` 决定兼容性，`appVersion` 仅用于展示。
- 用户设置中不再保存应用版本号，避免旧本地设置卡住旧版本显示。
- 导入和导出统计现在覆盖日志、伴侣、标签、周期事件、怀孕事件与内部快照。

## [0.1.1] - 2026-05-20

### Added
- 新增 CSV 导出，按天、性事件、自慰、运动、饮酒拆分为多份表格并打包下载。
- 新增 Markdown 导出，按天生成日记式摘要并附带近 7 日趋势。
- 导入流程新增预览页，展示数据版本、导出时间、包含的数据量和导入策略。
- 加密导出和加密导入可用于离线备份敏感数据。
- 自动安全快照新增“自动”标记，并保留最近 7 个自动安全快照。

### Changed
- 文件系统备份恢复改为先进入导入预览，再由用户确认合并或覆盖。
- 导入高版本数据时会拒绝导入，避免旧代码误读新数据结构。

### Fixed
- 备份恢复、数据修复和导入前的自动快照统一走安全快照流程。

## [0.1.0] - 2026-05-20

### Added
- 新增 PIN 应用锁，支持离开应用后自动锁定。
- 新增 WebAuthn 解锁支持，可使用设备生物识别能力解锁。
- 新增加密导出/导入，降低敏感备份文件外泄风险。
- 新增 iOS 安装引导，帮助用户把 PWA 添加到主屏幕。

### Changed
- 深色模式下的正文与卡片对比度更稳定。
- PWA 安装元数据更完整，安装体验更接近原生应用。

### Fixed
- 增加生理日常量测试，锁定凌晨归属规则，避免后续重构误改。

## [0.0.9] - 2026-05-19

### Added
- 统计页洞察按相关性、异常事件、行为模式和偏好趋势分组展示。
- Dashboard 新增 7 日趋势面板，展示硬度、睡眠、屏幕、自慰频次和运动变化。
- Dashboard 新增影响因子发现，帮助观察睡眠、饮酒、压力、运动和性活动负荷对硬度的影响。
- 性维度统计新增 XP 维度详情，展示记录数、标签出现次数和独立标签数。
- 运动和性爱记录支持智能默认值提示。

### Changed
- 日记编辑中的记录卡片统一到 `RecordCard` 展示。
- 性活动从生活 tab 中拆出，成为独立 tab。
- Toast 改为可堆叠队列，多条提示可同时自然展示。
