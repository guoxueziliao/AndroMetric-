# 硬度日记 - 用户使用手册

欢迎使用 **硬度日记** (Hardness Diary)。这是一款专为男性设计的健康追踪应用，旨在通过科学的数据记录与分析，帮助您了解身体状态、优化生活方式，并探索影响性健康的关键因素。

## 🛡️ 隐私声明 (重要)

**硬度日记采用“本地优先 (Local-First)”架构。**

*   **数据存储**: 您记录的所有数据（包括日记、伴侣档案、设置）仅存储在您当前使用的设备浏览器中 (`LocalStorage` / `IndexedDB`)。
*   **无云端同步**: 我们没有后台服务器，**不会**上传您的任何私密数据。
*   **数据安全**: 请务必定期使用“我的 -> 数据快照”或“导出”功能备份您的数据，以防清理浏览器缓存导致数据丢失。

## 🚀 快速入门

### 1. 每日晨间打卡
建议您在每天起床后尽快完成记录。
*   **晨勃状态**: 记录硬度 (1-5级) 和维持时间。这是评估男性心血管与神经系统健康的重要指标。
*   **睡眠质量**: 记录入睡与起床时间。应用会自动计算睡眠时长，并分析其对次日状态的影响。

### 2. 快速记录 (FAB)
在首页右下角点击 `+` 号悬浮按钮，可快速记录：
*   🌑 **睡眠**: 睡前点击，标记入睡时间。
*   ❤️ **性爱**: 快速添加一条性生活记录（稍后可补充详情）。
*   🖐️ **自慰**: 快速添加一条自慰记录。
*   🏃 **运动**: 记录实时运动（开始/结束）或步数。
*   😴 **午休**: 就像记录睡眠一样，点击开始午休，醒来点击结束。
*   🍺 **饮酒**: 快速记录饮酒量。

## ✨ 核心功能详解 (v0.0.6 更新)

### 🧬 全局时间轴 (Global Timeline)
**新功能**：在每一天的日记详情页底部，系统会自动将您当天的所有活动串联成一条完整的时间线。
*   **包含内容**: 睡眠起止、咖啡因摄入、运动、性活动、饮酒、自慰等。
*   **价值**: 帮助您直观回顾“一天的精力是如何流动的”，以及不同行为之间的时间关联（例如：喝咖啡是否太晚影响了入睡）。

### 💯 数据质量评分 (Quality Score)
为了鼓励记录的完整性，我们在日记顶部增加了实时评分环。
*   **基础分**: 记录了晨勃状态和睡眠时间即可及格。
*   **加分项**: 记录了心情、天气、运动、或详细的性活动细节。
*   **满分目标**: 尝试达到 100 分，这意味着您留下了极具分析价值的一天。

### 📅 日历与概览
首页以日历热力图形式展示您的记录情况。
*   **颜色含义**:
    *   🟩 绿色背景: 当天有记录且晨勃状态良好。
    *   🟨 黄色背景: 记录待完成。
    *   图标点: ❤️性爱 / 🖐️自慰 / 🍺饮酒 / 🎬看片 / 🏃运动
*   **操作**: 点击日期可查看当天的详细摘要或进行编辑。

### 🖐️ 自慰记录 2.0
我们彻底重构了自慰记录模块，现在它是您探索个人性癖 (XP) 的核心工具。
*   **Quick Log**: 点击开始后，只记录时间；结束后可选择“快速结案”或“补全详情”。
*   **XP 雷达**: 记录内容的**来源**、**平台**与**标签（XP）**，系统会自动生成六维雷达图。
*   **射精强度 (v0.0.6)**: 新增“射精量/力度”评级（1-5级），用于追踪前列腺健康与禁欲效果。
*   **贤者时间**: 记录射精后的心理（满足/空虚）与生理（精神/秒睡）状态。

### ❤️ 性爱日记
我们提供了极高自由度的性生活记录模块：
*   **互动流 (Interactions)**: 支持记录一次性生活中的多个“阶段”（如前戏、正戏）。
*   **动作链**: 按顺序记录发生的行为（如：口交 -> 传教士 -> 后入）。
*   **全局设定**: 记录避孕措施、自身状态（清醒/微醺）、是否射精及位置。

### ☕ 生活方式 (Lifestyle)
*   **饮酒**: 支持多种酒类换算为纯酒精克数，并记录醉意程度（微醺/断片）。
*   **咖啡因 (新)**: 记录咖啡、茶或能量饮料的摄入，追踪其对睡眠的影响。
*   **健康状态**: 记录生病时的**不适程度**（轻微/严重）、**具体症状**（如头痛、发烧）及用药情况。
*   **运动与午休**: 完整的运动类型库与午休梦境记录。

### 👥 伴侣档案
在“性生活”页面点击右上角“伴侣档案”进入。
*   **深度画像**: 记录伴侣的三围、敏感点、XP偏好、生理周期与关键纪念日。
*   **关系管理**: 区分固定、约会、露水或服务型伴侣，并统计与每位伴侣的互动频率与满意度。

### 🏷️ 标签管理与体检 (Tag System)
随着记录增多，标签可能会变得杂乱。我们在“我的 -> 设置”中提供了维护工具：
*   **标签体检**: 自动扫描您的 XP、事件与症状标签，发现命名不规范、语义冲突或长期未使用的标签。
*   **风险提示**: 当您创建如“人妻出轨”这种复合标签时，系统会建议拆分为“人妻（角色）”+“出轨（剧情）”，以保证统计准确性。

## ⚙️ 数据备份与迁移

由于数据仅存储在本地，**数据备份非常重要**。

1.  **数据快照**: 在“我的”页面，点击“创建快照”。这会在本地保存一份备份。
2.  **导出文件**: 点击“导出为通用 JSON”。这会生成一个 `.json` 文件，您可以将其发送到电脑或其他设备保存。
3.  **恢复数据**: 在新设备或清理缓存后，使用“从文件导入”功能读取之前的 JSON 文件即可恢复数据。

## ❓ 常见问题 (FAQ)

**Q: 为什么我有晨勃但是日历没变绿？**
A: 只有硬度达到一定等级（通常 >2级）且记录完整时，日历才会显示为健康的绿色。

**Q: 我晚上 1 点做的爱，应该记在今天还是昨天？**
A: 应用采用“生理日”逻辑。凌晨 3 点之前的活动，通常会被自动归类到“昨天”，以符合人的心理认知。

**Q: 标签体检里的“风险标签”是什么意思？**
A: 这意味着该标签可能包含多个维度的含义（如“黑丝足交”同时包含了装扮和行为），这会导致统计不准。建议拆分为两个独立标签。
