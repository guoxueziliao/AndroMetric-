# 硬度日记 - 用户使用手册 (v0.0.6)

欢迎使用 **硬度日记** (Hardness Diary)。这是一款专为男性设计的生理健康追踪与分析工具，旨在通过私密、客观的数据记录，帮助您建立生活方式与身体机能之间的科学联系。

## 🛡️ 隐私与安全原则

**硬度日记采用“本地优先 (Local-First)”架构。**

*   **数据主权**: 您的所有日记、伴侣档案、性癖偏好均仅存储在当前设备的浏览器数据库 (`IndexedDB`) 中。
*   **物理隔离**: 应用不设后台服务器，不会上传任何数据到云端。
*   **风险自担**: 请务必养成定期导出 JSON 备份或创建“数据快照”的习惯。清理浏览器缓存或重装系统会导致本地数据丢失。

---

## 🚀 核心记录指南

### 1. 晨间生理打卡（核心指标）
建议每日醒来后第一时间完成。
*   **硬度分级 (1-5级)**: 从“豆腐”到“铁棒”，客观评估海绵体充血质量。这是心血管健康与雄激素水平的“晴雨表”。
*   **维持时间**: 记录勃起消退的速度，反映神经系统的兴奋抑制平衡。
*   **睡眠质量**: 精确记录入睡与醒来时间。系统会自动分析“熬夜”或“睡眠不足”对次日硬度的定量影响。

### 2. 快速记录 (FAB)
通过首页右下角的 `+` 悬浮按钮，您可以捕捉实时的生理事件：
*   🌑 **睡眠计时**: 关灯时点击，醒来时结束，系统自动补全日期归档。
*   ❤️ **性爱/自慰**: 记录开始时间与时长，支持后续补充具体的互动链与素材。
*   🏃 **运动**: 记录训练强度与身体疲劳感。
*   🍺 **饮酒**: 支持多种酒类换算，追踪酒精对次日生理反应的抑制效应。

## ✨ v0.0.6 重磅功能详解

### 🧬 全局时间轴 (Global Timeline)
在日记详情页底部，系统会自动将您当天的所有零散活动串联。
*   **时间流视图**: 按照“起床 -> 咖啡 -> 运动 -> 性活动 -> 睡眠”的顺序排列。
*   **价值**: 帮助您直观发现“下午 4 点喝的咖啡是否影响了晚上 11 点的入睡”，以及“饮酒后多久出现了硬度下降”。

### 💯 数据质量评分 (Quality Score)
日记顶部新增“实时评分环”，用于衡量记录的科研/复盘价值。
*   **基础分**: 记录了硬度、持续时间和睡眠时间。
*   **完整分**: 记录了心情、压力、天气、地点以及详细的活动描述。
*   **目标**: 达到 100 分意味着这一天的数据具有极高的分析价值，能为“因子实验室”提供准确样本。

### 🏷️ 标签管理与体检 (Tag System)
随着记录增加，标签容易混乱。我们在“我的 -> 标签管理”提供了维护工具：
*   **标签体检**: 自动识别“复合标签”（如：黑丝足交，应拆分为“黑丝”+“足交”），确保统计维度单一、准确。
*   **语义校验**: 防止将平庸的“很爽”、“一般”作为标签，引导用户记录具体的“角色”、“行为”或“剧情”。

### 🖐️ 自慰记录 2.0 (XP 探索版)
*   **射精强度**: 新增 1-5 级“量与力度”评级，用于追踪前列腺健康状态及禁欲周期效果。
*   **贤者时间**: 记录射精后的心理（空虚/满足）与生理（秒睡/精神）反馈。
*   **素材溯源**: 将观看的内容数字化，分析哪些平台或题材最能激发您的潜能。

*   **因子实验室**: 系统会自动对比“高压力组”与“低压力组”的平均硬度差。

## ❓ 常见问题

**Q: 凌晨 2 点发生的记录算哪一天？**
A: 本应用采用“生理日”逻辑。凌晨 3 点前的活动归属于“昨天”；中午 12 点后的睡眠归属于“醒来后的那天”。

**Q: 导出 JSON 有什么用？**
A: 它是您唯一的数据备份。您可以将它保存在网盘或电脑上，在新设备上点击“导入”即可瞬间恢复所有历史记录。
