# 硬度日记 - 用户使用手册

欢迎使用 **硬度日记**。这是一款专为男性设计的健康追踪应用，旨在通过科学的数据记录与分析，帮助您了解身体状态、优化生活方式，并探索影响性健康的关键因素。

## 🛡️ 隐私声明 (重要)

**硬度日记采用“本地优先 (Local-First)”架构。**

*   **数据存储**: 您记录的所有数据（包括日记、伴侣档案、设置）仅存储在您当前使用的设备浏览器中 (`LocalStorage` / `IndexedDB`)。
*   **无云端同步**: 我们没有后台服务器，**不会**上传您的任何私密数据。
*   **数据安全**: 请务必定期使用“我的 -> 数据快照”或“导出”功能备份您的数据，以防清理浏览器缓存导致数据丢失。

## 🚀 快速入门

### 1. 每日晨间打卡
建议您在每天起床后尽快完成记录。
*   **晨勃状态**: 记录硬度 (1-5级) 和维持时间。这是评估男性心血管与神经系统健康的重要指标。
*   **睡眠质量**: 记录入睡与起床时间。应用会自动计算睡眠时长，并分析其对次日状态的影响。

### 2. 快速记录 (FAB)
在首页右下角点击 `+` 号悬浮按钮，可快速记录：
*   🌑 **睡眠**: 睡前点击，标记入睡时间。
*   ❤️ **性爱**: 快速添加一条性生活记录（稍后可补充详情）。
*   🖐️ **自慰**: 快速添加一条自慰记录。
*   🏃 **运动**: 记录实时运动（开始/结束）或步数。
*   😴 **午休**: 就像记录睡眠一样，点击开始午休，醒来点击结束。
*   🍺 **饮酒**: 快速记录饮酒量，支持多种酒类换算为纯酒精克数。

## ✨ 核心功能详解

### 📅 日历与概览
首页以日历热力图形式展示您的记录情况。
*   **颜色含义**:
    *   🟩 绿色背景: 当天有记录且晨勃状态良好。
    *   🟨 黄色背景: 记录待完成。
    *   图标点: ❤️性爱 / 🖐️自慰 / 🍺饮酒 / 🎬看片 / 🏃运动
*   **操作**: 点击日期可查看当天的详细摘要或进行编辑。

### 🖐️ 自慰记录 2.0 (全新升级)
我们彻底重构了自慰记录模块，现在它是您探索个人性癖 (XP) 的核心工具。记录界面分为四个维度：

#### 1. 🕒 时间 (Basic)
*   **开始时间**: 记录开始“施法”的时间。
*   **持续时长**: 记录从开始到射精（或结束）的总时长。

#### 2. 🧩 内容 (Content) - XP 雷达的核心
这是本次升级的重点，您的选择将直接生成统计页面的“XP 雷达图”。
*   **来源 (Source)**: 您在用什么施法？支持多选（如“视频+幻想”）。
    *   *选项*: 视频、直播、图片、文爱、回忆、幻想、音声 (ASMR) 等。
*   **平台 (Platform)**: 素材来自哪里？
    *   *选项*: Pornhub, Twitter, OnlyFans, Jable, 本地硬盘等。
*   **类型/XP (Category)**: **核心功能**。记录本次素材的关键词。
    *   *选项*: 巨乳、足控、人妻、NTR、颜射、SM 等。
    *   *智能排序*: 系统会自动将您最常选的标签排在前面。
*   **施法对象 (Target)**:
    *   **关联伴侣**: 可以直接点击下方头像，选择您“伴侣档案”中的人（适合幻想或看照片场景）。
    *   **临时对象**: 也可以手动输入网黄、明星或临时昵称。

#### 3. 🔧 操作 (Action)
*   **润滑剂**: 是否使用？类型是水溶性、油性还是唾液？
*   **边缘控制 (Edging)**: 记录是否进行了“快射出来时停下”的控精练习。
    *   *选项*: 无、一次边缘、多次边缘。
*   **辅助工具**: 记录是否使用了飞机杯、倒模、慢玩玩具等。

#### 4. 😊 结局 (Outcome)
*   **最终射精**: 有些练习可能以“不射精”结束（寸止），您可以取消勾选此项。
*   **愉悦强度**: 1-5 分的主观评分（从“无感”到“升天”）。
*   **备注**: 记录番号、链接或当下的特殊感受。

### ❤️ 性爱日记 (高级功能)
我们提供了极高自由度的性生活记录模块：
*   **互动流 (Interactions)**: 一次性生活可以包含多个“阶段”。例如，您可以添加多个阶段来记录“客厅前戏” -> “卧室正戏”的过程。
*   **动作链**: 在每个阶段中，按顺序记录发生的行为（如：口交 -> 传教士 -> 后入）。
*   **丰富的选项库**: 内置了大量主流的体位与行为标签，以及更丰富的地点（如楼梯、沙发）和道具选项。
*   **状态记录**: 记录当下的自身状态（如：清醒、微醺、醉酒、亢奋等）。
*   **细节标记**: 记录使用的服装 (Costumes)、玩具 (Toys)、是否射精、射精位置以及伴侣的高潮情况。

### 🏃 运动与午休
*   **运动追踪**: 支持记录运动类型（如跑步、撸铁）和时长。对于日常步行，可以直接记录总步数。
*   **午休管理**: 简单的开始/结束计时，自动计算时长，不再需要事后回忆填写。

### 👥 伴侣档案
在“性生活”页面点击右上角“伴侣档案”进入。
*   **档案管理**: 记录伴侣的三围、敏感点、开发度（如深喉等级）、XP偏好等。
*   **深度画像**: 
    *   **关系分类**: 区分固定、约会、露水或服务型伴侣。
    *   **生活习惯**: 记录吸烟、饮酒习惯。
    *   **背景与心理**: 记录家庭背景、最看重的价值观以及最讨厌的雷点。
    *   **反差刻画**: 记录伴侣在“日常生活中”与“床上”的反差形象。
*   **关联记录**: 在性爱日记中选择伴侣，可自动关联其档案信息。

### 📊 数据统计
*   **总览**:
    *   **趋势图**: 展示晨勃硬度与睡眠时长的 7 日移动平均线。
    *   **因子实验室**: 自动分析生活习惯（如饮酒、熬夜、运动）对**次日**晨勃硬度的影响。
*   **习惯 (Behavior) - 新增**:
    *   **性癖雷达 (XP Radar)**: 根据您在自慰记录中填写的“类型/Category”标签，自动生成六维雷达图，直观展示您的性兴奋点分布。
    *   **活跃时段**: 分析您在一天中哪个时段（早/中/晚/夜）最活跃。

## ⚙️ 数据备份与迁移

由于数据仅存储在本地，**数据备份非常重要**。

1.  **数据快照**: 在“我的”页面，点击“创建快照”。这会在本地保存一份备份。
2.  **导出文件**: 点击“导出为通用 JSON”。这会生成一个 `.json` 文件，您可以将其发送到电脑或其他设备保存。
3.  **恢复数据**: 在新设备或清理缓存后，使用“从文件导入”功能读取之前的 JSON 文件即可恢复数据。

## ❓ 常见问题 (FAQ)

**Q: 为什么我有晨勃但是日历没变绿？**
A: 只有硬度达到一定等级（通常 >2级）且记录完整时，日历才会显示为健康的绿色。

**Q: 我晚上 1 点做的爱，应该记在今天还是昨天？**
A: 应用采用“生理日”逻辑。凌晨 3 点之前的活动，通常会被自动归类到“昨天”，以符合人的心理认知。

**Q: XP 雷达图怎么全是空的？**
A: 雷达图基于“自慰记录”中的“类型/XP”标签生成。您需要多记录几次，并确保选择了如“巨乳”、“足控”等标签，雷达图才会出现。
