# 硬度日记 (Hardness Diary) v0.0.6

**硬度日记** 是一款专为男性设计的隐私优先、数据驱动的健康追踪应用。它不仅记录数据，更致力于通过科学的量化分析（如晨勃质量、睡眠周期、生活习惯关联），帮助您找回身体的最佳状态。

## 📚 文档指南

*   **[📖 用户使用手册 (User Manual)](USER_MANUAL.md)**: 面向用户的功能介绍、隐私说明及常见问题解答。

## v0.0.6 新特性

*   **全局时间轴**: 将睡眠、运动、咖啡因与性活动串联为可视化的生活流。
*   **数据质量评分**: 实时反馈记录完整度，鼓励高质量的数据积累。
*   **标签健康体系**: 包含自动体检与清洗建议，防止长期记录导致的语义混乱。
*   **深度自慰记录**: 新增射精强度（量/力）与贤者时间身心状态评估。
*   **咖啡因追踪**: 探究兴奋剂摄入对睡眠与晨勃的长期影响。

## 主要功能

*   **核心追踪**: 晨勃硬度、维持时间、睡眠质量与梦境记录。
*   **性爱日记**: 支持多伴侣、多阶段互动、体位/道具/着装细节的全维度记录。
*   **生活方式**: 饮酒（纯酒精换算）、咖啡因、运动、压力与心情。
*   **隐私安全**: 本地优先 (Local-First)，数据仅存设备，支持 JSON 导出备份与快照回滚。
*   **数据洞察**: 
    *   **趋势分析**: 移动平均线揭示身体变化趋势。
    *   **XP 雷达**: 基于标签系统的性偏好六维分析。
    *   **因子相关性**: 自动分析熬夜、酒精等因素对硬度的具体影响。
*   **伴侣档案**: 建立详细的伴侣偏好与身体密码数据库。

## 开发指南

本项目使用 React + TypeScript + Tailwind CSS 构建。

⚠️ **重要提示**: 在进行任何代码修改前，请务必阅读 **`USER_MANUAL.md`**。该文档详细记载了项目的核心架构、日期判定逻辑（Physiological Day）、数据迁移规则以及组件规范。

### 快速开始

```bash
npm install
npm run dev
```

### 项目结构

```
src/
├── components/      # UI 组件 (Modals, Charts, Forms)
├── contexts/        # React Context (DataContext, ToastContext)
├── hooks/           # Custom Hooks (useLogs, useLocalStorage)
├── services/        # 业务逻辑层 (Storage, Logger, Plugins)
├── utils/           # 工具函数 (Migration, Stats, Helpers)
├── types.ts         # TypeScript 类型定义
└── db.ts            # Dexie 数据库配置
```
