
# v0.0.6 · XP 标签维度判定规则（Final）

## 0. 本规范的目的（写给未来的你）

当你在新建或整理 XP 标签时，不要凭感觉判断放哪一类，而是：

**只回答一个问题：**
**这个词描述的是「刺激来源的哪一层信息」？**

本规则用于：
- 防止 XP 标签维度混乱
- 防止一个标签“什么都像”
- 防止半年后自己也解释不清

---

## 1. XP 六维度的“唯一判定问题”

每个 XP 标签 **只能回答其中一个问题**。
如果同时回答 ≥2 个 → **高风险复合标签**，必须拆分。

| 维度 | 判定问题（只问这一个） |
| :--- | :--- |
| **角色与关系** | 她是谁？和我是什么关系？ |
| **外貌与身体** | 她长什么样？身体哪部分刺激我？ |
| **装扮与道具** | 她穿了什么 / 用了什么？ |
| **玩法与行为** | 发生了什么性行为或刺激动作？ |
| **剧情与情境** | 这是一个什么关系/权力/心理情境？ |
| **形式与风格** | 内容是“以什么媒介/表现方式”呈现的？ |

⚠️ **只能回答一个问题**。一旦“顺便还能回答另一个”，就要拆。

---

## 2. 各维度【允许 / 禁止】的明确边界

### 2.1 角色与关系（WHO）
*   **允许**：人物身份、社会角色、关系结构（不涉及身体、不涉及行为、不涉及剧情走向）
*   ✅ 人妻、老师、学生、护士、母亲、姐姐、女上司
