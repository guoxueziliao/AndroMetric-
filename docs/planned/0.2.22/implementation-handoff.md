# 0.2.22 实现交接摘要

## 版本定位

0.2.22 是设置与偏好整理 v1，范围档位 A（纯信息架构整理）。

`AppSettings` 结构克制无需改；杂乱在 `MyView.tsx` 的设置面板分区与标题样式。本版只整理分组、归属、样式，行为完全等价。

## 已确认决策

- 范围 A：纯信息架构整理。
- 不改 `AppSettings` 字段、类型、默认值。
- 不动存储、不新增 schema / migration。
- 不拆 `MyView.tsx`（B 档暂不做）。
- 不收束偏好存储访问层（C 档暂不做）。
- 只复用现有 token，不引入新视觉。
- 任何整理必须行为等价。

## 推荐实现顺序

1. 真实代码校准：列出设置 Modal 全部分区与项。
2. 定稿目标分组。
3. JSX 重排到目标分组，props / handler 不变。
4. 统一标题与项样式。
5. 命名校准。
6. 行为等价回归。

## 停线项

- 必须拆组件才能完成（升级 B）。
- 必须改字段 / 存储（升级 C / 需 schema）。
- 需要新增设置项或新功能。
- 需要新视觉风格。
- 需要改备份 / 导出 JSON 字段名。

## 验收底线

- 分区与分组契约一致，标题样式统一。
- 每项只归一个分区。
- 行为等价：功能 / 读写 / 默认值不变。
- `settings.ts` 与 `appConfig.ts` 的 `git diff` 为空。
- 危险操作仍独立、显著、需确认。
