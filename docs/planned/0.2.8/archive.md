# 0.2.8 候选能力归档

> 本文记录 0.2.8 讨论中暂不进入本版的能力，避免后续反复打开同一问题。

## 人群基准

不进入 0.2.8。

原因：

- 容易被误读为医学正常值。
- 会制造比较压力。
- 需要来源、年龄段、群体和统计口径说明。

## 医学异常检测

长期不做。

原因：

- 产品不是医疗诊断工具。
- 本地记录样本不足以支持诊断。
- 容易造成误导和焦虑。

## 未来表现预测

不进入 0.2.8。

原因：

- 当前数据只适合复盘和观察。
- 预测会暗示系统知道未来表现。
- 容易滑向强因果。

## 长期 Baseline Cache

默认不进入 0.2.8。

原因：

- 派生计算可以先满足需求。
- cache 会带来失效、迁移和导入导出契约。
- 若真实性能不足，再单独讨论。

## 外部 AI 分析

不进入 0.2.8。

原因：

- 与本地隐私基线冲突。
- 成人健康和关系数据不应上传。

## 分享图或对外报告

不进入 0.2.8。

原因：

- 长期趋势包含高度私密的健康和成人行为信息。
- 当前产品不鼓励社交传播。
