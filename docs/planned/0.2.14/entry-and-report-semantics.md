# 0.2.14 入口与报告语义

## 入口语义

0.2.14 要减少“点击后不知道发生了什么”的入口。

入口文案必须表达动作：

- 查看回顾。
- 查看目标。
- 开始目标。
- 取消目标。
- 归档目标。
- 导出文件。

## 报告语义

- “报告”默认表示查看，不默认表示导出。
- UI 必须区分“查看回顾”和“导出文件”。
- 周报 / 月报入口点击后优先进入只读查看页。
- 导出是查看页里的二级动作。
- 导出按钮必须明确格式，例如“导出 CSV”或“导出 JSON”。

## 降级策略

如果实现暂时没有查看页，文案至少要写成“导出报告文件”，不能只写“报告”。

## 不做

- 不新增报告算法。
- 不做对外医疗报告。
- 不做分享图。
