# 0.2.14 验收与交接

## 推荐实现顺序

1. 代码校准：确认复盘入口、报告入口、训练目标入口和色情使用入口真实位置。
2. 复盘窗口：让 7 天、14 天、30 天、周报、月报有可感知差异。
3. 训练目标管理：防重复开始，补取消 / 归档 / 隐藏入口。
4. 报告语义：查看和导出拆开。
5. 色情使用入口：统一入口和文案。
6. Safety / Privacy 审计：确保不扩大成人内容和训练能力边界。

## 验收底线

- 用户能看出 7 天、14 天、30 天、周报、月报的差异。
- 同一条训练建议不能重复开始多个目标。
- 误创建的无历史目标可以取消。
- 有历史的目标可以归档或隐藏。
- 目标列表和详情都有管理入口。
- 周报 / 月报入口默认查看，不默认导出。
- 导出动作明确写出格式。
- 色情使用有稳定入口，不只依赖日期表单。
- “生活 / 看片”不形成独立记录口径。
