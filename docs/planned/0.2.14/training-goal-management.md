# 0.2.14 训练目标管理契约

## 三层语义

- 训练建议：系统给出的建议，不是目标。
- 训练目标：用户点击“开始”后创建的进行中目标。
- 历史记录：目标执行、check-in 和过往状态。

## 开始目标

- 同一条建议已开始后，不再显示“开始”。
- 已开始建议显示“已开始”或“查看目标”。
- 重复点击不能创建多条重复目标。

## 取消 / 归档 / 隐藏

- 刚误创建且没有历史记录的目标允许“取消目标”。
- 已有 check-in 或历史记录的目标优先“归档目标”。
- 用户只是不想在当前列表看到时，可以隐藏。
- 硬删除只允许在没有历史记录且当前数据结构支持时出现。

## 入口要求

- 目标列表必须有管理入口。
- 目标详情必须有管理入口。
- 不能只藏在复盘入口或建议卡片里。

## 停下来重谈

如果取消、归档、隐藏或删除需要新增 schema / migration，应停止本版实现并重新讨论。
