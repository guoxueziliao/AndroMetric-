# 0.2.17 实现交接摘要

## 版本定位

0.2.17 是高级筛选与自定义回看 v1。

它面向长期数据回看，让用户按时间、行为、关系、健康项目和数据质量组合筛选记录。

## 已确认决策

- 只做筛选回看。
- 不做全局搜索 / 快速跳转。
- 不做全文搜索。
- 不做 AI 问答。
- 不保存自定义筛选视图。
- 默认不新增 schema / migration。

## 推荐实现顺序

1. 代码校准：确认日志、成人行为、关系、健康项目、训练目标和状态页入口。
2. 筛选条件：实现时间、行为、关系、健康项目和数据质量维度。
3. 组合规则：同维度多选，不同维度取交集。
4. 结果页：展示条件摘要、命中数量、时间范围、记录列表和事实摘要。
5. 可信度：接入 0.2.16 的样本限制和可信度说明。
6. Safety / Privacy 审计。

## 停线项

- 需要保存筛选视图。
- 需要新增数据表或 migration。
- 需要全文搜索。
- 需要全局搜索 / 快速跳转。
- 需要 AI 问答。
- 需要报表中心或对外分享。

## 验收底线

- 用户可以组合多维条件回看长期记录。
- 结果页清楚显示当前条件和命中数量。
- 结果太少时提示放宽条件，不生成结论。
- 成人行为筛选不变成内容管理器。
- 不新增 schema / migration。
