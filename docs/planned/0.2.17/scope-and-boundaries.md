# 0.2.17 范围与边界

## 核心定位

0.2.17 让长期用户主动回看自己的长期数据。

用户可以按多个条件组合查看记录，而不是只能看固定周报、月报或趋势页。

## 做什么

- 多维筛选。
- 自定义回看结果页。
- 筛选后摘要。
- 入口连接到相关趋势、复盘和记录详情。

## 默认边界

- 默认只读。
- 默认不新增 schema / migration。
- 默认不保存筛选视图。
- 默认不新增分析算法。
- 已确认不做全局搜索 / 全文搜索。

## 不做什么

- 不做全局搜索。
- 不做全文搜索。
- 不做 AI 问答。
- 不做报表中心。
- 不做对外分享。
- 不做医学解释。
