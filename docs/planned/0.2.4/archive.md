# 0.2.4 候选方向归档

> 本文从 0.2.4 入口拆出，保存早期候选方向，作为后续版本讨论参考。

## 候选方向

### 方向 A：养成系正式化

把 0.2.3 的复盘和弱相关观察继续推进为轻量养成系统。

可能内容：

- 记录质量成长。
- 恢复习惯成长。
- 硬度稳定性成长。
- 做爱表现稳定性成长。
- 关系沟通成长。
- 数据可信度成长。

不应做：

- 性次数挑战。
- 射精次数挑战。
- 多伴侣数量挑战。
- 色情使用时长挑战。
- 鼓励高风险行为的目标。

### 方向 B：性表现训练

围绕做爱过程中的硬度、时长、射精控制、满意度、疲劳和恢复，设计训练与反馈。

可能内容：

- 硬度稳定性训练。
- 射精控制 / 边缘控制后的恢复观察。
- 做爱时长稳定性。
- 前戏 / 唤起 / 色情刺激参与的影响复盘。
- 睡眠、运动、酒精、压力对表现的建议。

边界：

- 不做医学诊断。
- 不宣称治疗 ED。
- 不输出强因果。
- 不做羞辱或 PUA 式话术。

### 方向 C：恢复训练

把高负荷性行为、自慰、色情使用、射精、边缘控制和睡眠恢复串起来。

可能内容：

- 高负荷后恢复窗口。
- 连续疲劳提示。
- 睡眠补偿。
- 酒精 / 咖啡因影响提醒。
- 低硬度期的恢复建议。

### 方向 D：关系与伴侣维度深化

在不做伴侣评分和排名的前提下，支持多伴侣、多性别伴侣和多关系形态下的复盘与沟通。

可能内容：

- 关系形态记录。
- 伴侣沟通记录。
- 事后反馈。
- 边界和偏好记录。
- 不同关系上下文下的表现与满意度复盘。

明确不做：

- 伴侣排名。
- 伴侣优劣判断。
- 关系审判。
- 自动判断“适不适合某伴侣”。
