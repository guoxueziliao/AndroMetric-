# 0.2.11 入口与回看

> 本文定义经验卡从哪里保存，以及后续如何回看。

## 保存入口

允许入口：

- 观察计划到期回看页。
- 观察计划历史详情。
- 用户手动创建。

主入口应是观察计划回看页。

## 保存流程

保存前展示：

- 观察窗口。
- 事实摘要。
- limitations。
- 用户可编辑的标题和心得。

系统不得自动保存经验卡。

## 回看入口

允许回看：

- 0.2.5 / 0.2.10 历史附近。
- 后续历史入口。
- 观察计划历史详情。

不新增顶级导航。

## 筛选

第一版允许按：

- 指标。
- 上下文类型。
- 日期。
- 来源观察计划。

不做搜索全文和复杂知识库。

## 停下来重谈

- 需要自动保存。
- 需要全局知识库入口。
- 需要独立经验知识库。
- 需要分享或导出经验卡。
- 需要把经验卡用于自动建议。
