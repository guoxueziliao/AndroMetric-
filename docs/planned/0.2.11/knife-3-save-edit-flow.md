# 0.2.11 刀 3 保存入口与编辑确认流

> 本文定义经验卡的保存入口、编辑确认和取消边界。

## 入口

主入口：

- 观察计划到期回看页。

辅助入口：

- 观察计划历史详情。

不新增顶级入口。

## 保存前

保存前展示：

- 关联观察窗口。
- factSummary。
- limitations。
- 可编辑 title。
- 可编辑 userReflection。

用户必须手动确认保存。

## 编辑规则

- 用户可编辑 title。
- 用户可编辑 userReflection。
- factSummary 可复制但不作为系统结论。
- limitations 不可隐藏。

## 取消规则

- 取消不保存草稿，除非已有本地 draft 机制。
- 不用 toast 催促用户保存。
- 不把未保存视为损失或失败。

## 验收

- 不自动保存。
- 不自动改写用户心得。
- 保存确认不使用“规律成立”等文案。
- 取消流程无压力文案。
