# 0.2.11 刀 4 经验回看与筛选

> 本文定义经验卡历史列表、筛选和详情回看。

## 列表

列表展示：

- 标题。
- 日期窗口。
- 关联指标。
- 上下文标签。
- 一句用户心得摘要。

不展示：

- 证明强度。
- 成功率。
- 医学解释。
- 伴侣或色情敏感细节。

## 筛选

第一版支持：

- 指标。
- 上下文类型。
- 日期范围。
- 来源观察计划。

不做：

- 全文搜索。
- 自动聚类。
- 相似经验推荐。

## 详情

详情展示：

- factSummary。
- userReflection。
- limitations。
- 关联观察窗口。
- 回到来源记录的入口。

详情不自动生成建议或新计划。

## 验收

- 不新增顶级导航。
- 不把经验卡包装成知识库。
- 不展示敏感全文。
- 不自动推荐下一步。
