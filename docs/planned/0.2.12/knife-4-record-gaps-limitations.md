# 0.2.12 刀 4 记录缺口与限制说明

> 本文定义阶段回顾如何展示缺失记录、样本不足和隐藏章节。

## 缺口类型

需要展示：

- 缺记录日期。
- 样本不足指标。
- 缺少来源模块。
- 被隐藏章节。
- 只读派生限制。

不展示：

- 记录质量分。
- 用户评分。
- 达标 / 不达标。
- 医学准确率。

## 章节缺口

每个章节可以附带：

- 来源名称。
- 样本数量。
- 缺失说明。
- confidence。
- limitations。

如果章节来源未落地，隐藏章节并在记录缺口里说明“该来源尚未启用”。

## 文案规则

允许：

- “记录较少，本章节仅展示事实。”
- “部分来源尚未启用。”
- “缺口会影响回顾完整性。”

避免：

- “无法判断你的健康。”
- “数据质量差。”
- “请补齐记录。”
- “本报告不准确。”

## 验收

- 用户能知道为什么某章节没有出现。
- 缺口提示不会制造压力。
- 缺口不被写成医学可靠性。
- limitations 随回顾结果一同展示。
