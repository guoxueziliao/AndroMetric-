# 0.2.10 触发与入口

> 本文定义观察计划从哪里进入，以及哪些入口不能自动创建计划。

## 主入口

允许从以下位置进入：

- 0.2.9 解释卡片详情。
- 0.2.8 个人常态指标详情。

入口文案应是：

- “观察 7 天。”
- “加入短期观察。”
- “记录一段时间再看。”

不写成：

- “开始治疗。”
- “立即改善。”
- “挑战自己。”
- “必须完成。”

## 创建确认

创建前必须展示：

- 观察主题。
- 观察窗口。
- 需要关注的记录字段。
- 不做诊断或保证结果的说明。

用户必须手动确认。

## Dashboard

Dashboard 只允许轻提示：

- “有一个观察计划进行中。”
- “观察计划即将到期，可回看事实总结。”

不展示敏感主题、色情内容、伴侣或周期细节。

## 空状态

空状态允许：

- “还没有观察计划。”
- “可以从解释卡片中手动创建。”

不诱导创建计划。

## 停下来重谈

- 需要自动创建。
- 需要默认开启通知。
- 需要 Dashboard 展示敏感主题。
- 需要从伴侣页面创建监督型计划。
- 需要独立观察计划中心。
