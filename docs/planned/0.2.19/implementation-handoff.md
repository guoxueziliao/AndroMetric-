# 0.2.19 实现交接摘要

## 版本定位

0.2.19 是个人指标自定义与权重 v1。

它给长期用户一个“我现在更关心什么”的配置层，影响展示和默认优先级，不改变原始数据和健康结论。

## 已确认决策

- 第一版只做三档轻量分组。
- 三档为：重点关注 / 普通显示 / 弱化隐藏。
- 不做 1-100 精细权重。
- 不做医学权重。
- 不做 AI 自动个性化。
- 默认不新增 schema / migration。

## 推荐实现顺序

1. 真实代码校准：找出现有指标、设置结构、首页、状态页、筛选回看和洞察入口。
2. 建立轻量偏好模型。
3. 做设置入口和分组操作。
4. 将偏好接入首页 / 状态页排序。
5. 将偏好接入 0.2.17 筛选回看的默认维度。
6. 将偏好接入洞察提示优先级。
7. 回归检查弱化隐藏不丢数据。

## 停线项

- 现有设置结构无法保存三档偏好。
- 需要修改数据库 schema。
- 需要设计数值权重或权重算法。
- 需要医学解释或风险评分。
- 需要重新定义指标体系。

## 验收底线

- 偏好可配置、可修改、可恢复默认。
- 三档偏好能被多个模块读取。
- 首页、状态页、筛选回看至少各有一个可见变化点。
- 弱化隐藏不等于删除。
- 文案只表达个人关注重点，不表达医学判断。
