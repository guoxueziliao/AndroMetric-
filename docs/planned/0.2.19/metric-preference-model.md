# 0.2.19 指标偏好模型

## 分组模型

第一版只提供三档分组。

| 分组 | 含义 | 展示规则 |
|---|---|---|
| 重点关注 | 用户当前最想跟踪的指标 | 首页、状态页、筛选回看、洞察解释优先展示 |
| 普通显示 | 仍有价值但不需要置顶 | 按现有默认逻辑展示 |
| 弱化隐藏 | 暂时不关心或容易干扰判断 | 默认折叠、靠后或从重点摘要中移除 |

## 默认值

- 未配置时，所有指标保持普通显示。
- 重点关注不应过度限制数量，但界面可以提示“过多会降低聚焦感”。
- 弱化隐藏不代表删除数据，也不代表以后不再参与必要的完整记录查看。

## 可配置对象

优先从现有指标和模块中挑选，不在 0.2.19 发明新健康指标。

- 训练相关指标。
- 睡眠、精力、情绪、压力等状态指标。
- 色情使用和性健康相关指标。
- 关系上下文相关指标。
- 健康项目、补剂和周期相关指标。
- 长期趋势、阶段回顾和自定义回看的维度。

## 规则

- 一个指标同一时间只能处于一个分组。
- 分组是个人偏好，不是健康判断。
- 分组变化不回写历史记录。
- 未来如果要做数值权重，必须另开版本讨论。
