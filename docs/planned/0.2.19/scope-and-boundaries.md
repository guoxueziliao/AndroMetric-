# 0.2.19 范围与边界

## 核心定位

0.2.19 是个人关注重点层。

它让用户把长期使用后真正关心的指标排在前面，把暂时不关心但仍保留的数据弱化展示。

## 做什么

- 指标显示偏好。
- 三档轻量分组：重点关注 / 普通显示 / 弱化隐藏。
- 首页和状态页的展示优先级。
- 高级筛选和自定义回看的默认维度优先级。
- 洞察和提示的解释优先级。

## 默认边界

- 默认只影响展示、排序和默认入口。
- 默认不修改历史记录。
- 默认不新增 schema / migration。
- 默认不改变既有算法结论。
- 默认不保存复杂权重曲线。

## 不做什么

- 不做 1-100 精细权重。
- 不做医学风险加权。
- 不做 AI 自动个性化。
- 不做成瘾、疾病或疗效判断。
- 不做全局搜索 / 全文搜索。
- 不做隐私模糊 / 屏幕遮挡。
- 不做 Markdown 导出。
