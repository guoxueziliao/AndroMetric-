# 0.2.9 刀 4 UI 与 Dashboard

> 本文定义解释层的 UI 入口、卡片展示和 Dashboard 边界。

## 入口

主入口：

- 0.2.8 个人常态 / 长期趋势详情内。

辅助入口：

- Dashboard 轻提示，只跳转到 0.2.8 个人常态详情。

不做：

- 顶级导航。
- 独立解释层工作台。
- 独立解释报告。
- 分享页面。

## 首屏

建议结构：

1. 变化指标摘要。
2. 解释卡片列表。
3. 记录缺口。
4. 限制说明。

解释卡片不合成总分，不显示原因排名。

## 卡片内容

每张卡片展示：

- 上下文名称。
- 观察窗口。
- 温和 message。
- sampleSize。
- confidence。
- limitations。

详情层可以展示相关日期，但不展示 notes 全文。

## Dashboard

Dashboard 只允许：

- “有变化可回看。”
- “近期记录不足，暂不解释变化。”
- “可以在统计中查看变化附近的上下文。”

Dashboard 不展示：

- 色情使用细节。
- 伴侣 / 周期细节。
- 偏离原因。
- 强行动建议。

## 验收

- UI 只读。
- 不自动创建目标。
- 不新增顶级导航。
- 不新增独立工作台。
- Dashboard 不泄露敏感上下文。
- 空状态不制造失败感。

## 停下来重谈

- 需要独立报告页。
- 需要 Dashboard 展示具体敏感上下文。
- 需要一键创建目标。
- 需要分享图。
