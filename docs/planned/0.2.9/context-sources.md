# 0.2.9 上下文来源

> 本文列出解释层可以读取的上下文来源，以及依赖版本未落地时的隐藏规则。

## 可直接读取

来自 logs / StatsEngine：

- 睡眠时长和睡眠质量。
- 压力等级。
- 运动时长。
- 酒精。
- 屏幕时间。
- 天气 / 地点。

来自 adult behavior events：

- sex load。
- 色情使用频率 / 时长。
- 满意度。
- fatigueAfter。
- recoveryFeeling。

来自 0.2.3 review：

- 窗口 facts。
- 样本量。
- confidence。
- 记录缺口。

## 条件读取

0.2.5 真实落地后：

- 目标状态。
- check-in 历史。
- 目标 category。

0.2.7 真实落地后：

- 关系情境。
- 沟通反馈。
- 周期性生活关怀上下文。
- 不适、恢复和边界偏好。

## 隐藏规则

- 数据源不存在：隐藏对应上下文。
- 样本不足：只展示事实。
- 关系 / 周期内容缺失：不推断。
- 旧字段含义不清：加入 limitations。

## 第一版优先级

第一版优先：

1. 睡眠 / 压力。
2. 性负荷 / 恢复。
3. 色情使用。
4. 运动。

目标历史、关系上下文和周期上下文不抢占首屏。

## 禁止读取

- notes 全文。
- 色情内容标题、平台名或 URL。
- 伴侣私密备注全文。
- 任何外部数据源。
