# 0.2.17 高级筛选与自定义回看 v1

> 0.2.17 面向长期数据，允许用户按时间、行为、关系、健康项目等条件组合筛选并回看。它不是全局搜索，也不是新分析模型，而是长期数据的可控回看入口。

## 当前状态

- 状态：执行草案 / 待实现。
- 前置：0.2.16 长期使用体验与洞察校准 v1 已收口为执行草案。
- 当前实现线：0.2.7 为下一条候选线。
- 主方向：让长期用户能主动选择“我要看哪一段、哪类行为、哪个关系上下文、哪些健康项目相关记录”。
- 用户确认：只做筛选回看，不做全局搜索 / 全文搜索。
- 默认决策：自定义筛选视图不保存，先做临时视图，避免新增 schema / migration。

## 初步范围

- 时间筛选：最近 7 / 14 / 30 / 90 / 180 天、自定义区间、自然周 / 月。
- 行为筛选：睡眠、运动、酒精、咖啡因、性生活、自慰、色情使用。
- 关系筛选：伴侣、关系上下文、女性性健康关怀相关记录。
- 健康项目筛选：补剂、健康项目、观察计划和训练目标。
- 质量筛选：有完整记录、缺关键字段、可信度较低的窗口。
- 自定义回看：组合条件后查看记录列表、摘要和相关趋势入口。

## 明确不做

- 不做全局搜索 / 快速跳转。
- 不做全文搜索。
- 不做 AI 问答。
- 不新增分析大模型。
- 不做医疗报告。
- 不做分享图或对外导出报告。
- 默认不新增 schema / migration；保存筛选视图若需要持久化，必须先停下来确认。

## 已定边界

- 0.2.17 是长期数据回看工具，不是新的洞察算法。
- 筛选结果必须尊重 0.2.16 的可信度和样本限制。
- 成人行为筛选只用于个人回看，不做内容收藏或成瘾判定。
- 自定义回看默认是临时视图，不保存为固定视图。
- 0.2.17 排在 0.2.16 之后、0.2.18 之前。

## 暂不做

- 不在本窗口改产品代码。
- 不把 0.2.17 扩成报表中心。
