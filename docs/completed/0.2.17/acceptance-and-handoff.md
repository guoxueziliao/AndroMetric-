# 0.2.17 验收与交接

## 推荐实现顺序

1. 代码校准：确认日志、成人行为、关系、健康项目、训练目标和状态页数据入口。
2. 筛选模型：实现只读筛选条件和组合规则。
3. 结果页：展示命中记录、时间范围、列表和事实摘要。
4. 可信度：接入 0.2.16 样本限制和可信度说明。
5. 入口：从状态 / 复盘 / 长期趋势进入。
6. Safety / Privacy 审计。

## 验收底线

- 可以组合时间、行为、关系和健康项目筛选。
- 筛选结果显示条件摘要和命中数量。
- 结果太少时不生成结论。
- 不新增全局搜索。
- 不新增全文搜索。
- 不新增 schema / migration。
- 成人行为筛选不变成内容管理器。
- 自定义筛选视图默认不保存。

## 开发会话必读

- 先读 [`../plan-0.2.17.md`](../plan-0.2.17.md)。
- 再读本目录全部专题文档。
- 实现前必须校准真实日志、成人行为、关系、健康项目和训练目标数据入口。
- 若需要保存筛选视图或新增持久化，停下来重谈。

## 停下来重谈

- 需要保存自定义视图。
- 需要新增数据表或 migration。
- 需要全文搜索。
- 需要全局搜索 / 快速跳转。
- 需要 AI 问答。
- 需要新增报表中心。
