# 0.2.14 范围与边界

## 核心定位

0.2.14 解决“用户点进去不知道发生了什么、入口分散、误操作难处理”的问题。

它不是新模型版本，而是记录、复盘、报告和训练目标的入口收束版本。

## 做什么

- 复盘 / 统计窗口差异可感知。
- 训练建议和训练目标动作语义清楚。
- 重复开始目标可避免，误创建目标可处理。
- 报告查看和导出文件分开。
- 色情使用入口统一到成人行为记录主线。

## 默认边界

- 默认不新增 schema / migration。
- 优先处理入口、文案、状态和管理动作。
- 如果当前数据结构不支持目标取消 / 归档 / 隐藏 / 删除，应停下来重谈。

## 不做什么

- 不新增训练数据模型。
- 不新增复盘算法。
- 不做完整成人内容管理器。
- 不做色情内容收藏器。
- 不做 AI 教练、课程或训练游戏。
- 不新增后端、账号、云同步或外部 API。
