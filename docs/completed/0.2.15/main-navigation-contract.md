# 0.2.15 主导航归属

## 当前主导航

当前代码主导航是：

- 日历
- 状态
- 性生活
- 我的

0.2.15 第一版默认保留这四个顶级入口。

## 日历

职责：今天、某一天、快速记录和当日生活流。

应放：

- 每日记录。
- 快速记录。
- 今日提醒和待处理项。
- 某一天的时间线和编辑入口。

不应放：

- 长期趋势详情。
- 设置和数据维护。
- 训练目标完整管理。

## 状态

职责：统计、复盘、趋势、训练、观察计划、经验沉淀和阶段回顾。

应放：

- 当前状态。
- 历史统计。
- 复盘窗口。
- 周报 / 月报查看。
- 个人常态和长期趋势。
- 观察计划。
- 训练建议、训练目标和训练历史。
- 个人经验沉淀。
- 阶段回顾。

不应放：

- 单日表单主编辑。
- 伴侣资料维护主入口。
- 备份、导入导出和数据健康主入口。

## 性生活

职责：成人行为、关系上下文和伴侣相关体验。

应放：

- 性生活记录和成人行为时间线。
- 自慰 / 色情使用相关记录的主线入口或聚合入口。
- 伴侣上下文在成人行为中的使用。
- 女性性健康关怀和关系上下文的产品入口。

不应放：

- 色情内容收藏。
- 自动内容识别。
- 与成人行为无关的普通健康项目管理。

## 我的

职责：设置、数据、安全、管理类入口。

应放：

- 设置与数据。
- 备份、恢复、导入导出。
- 数据健康。
- 标签管理。
- 伴侣资料管理。
- 关于、版本、更新历史。

不应放：

- 日常高频记录主入口。
- 复盘和趋势主入口。
- 训练建议主入口。

## 快捷入口规则

同一能力可以有快捷入口，但只能有一个主归属。

例如：色情使用可以从日历某日快速记录，也可以从性生活进入，但两者必须写入同一数据结构，并回到同一成人行为口径。
