## 横向归属

- 记录动作优先归日历。
- 分析、复盘、趋势和训练优先归状态。
- 成人行为、伴侣和关系上下文优先归性生活。
- 设置、数据、安全和资料维护优先归我的。

## 冲突处理

- 如果一个入口同时像“记录”和“分析”，主归属按用户当前动作判断。
- 如果用户是“现在记录”，归日历。
- 如果用户是“回看和理解”，归状态。
- 如果用户是“亲密关系或成人行为”，归性生活。
- 如果用户是“管理基础资料或数据”，归我的。
