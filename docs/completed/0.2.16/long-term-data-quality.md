# 0.2.16 长期数据质量

## 目标

长期用户需要知道哪些数据能支撑洞察，哪些数据只是零散记录。

## 可展示内容

- 总记录天数。
- 最近 30 / 90 / 180 天记录覆盖率。
- 关键字段覆盖率：硬度、睡眠、情绪、运动、酒精、咖啡因、成人行为。
- 长期缺失字段。
- 近期记录突然变少的字段。
- 长期几乎不变的字段。
- 可能无效或低价值的字段。

## 字段分组

- 基础状态：硬度、睡眠、情绪、疲劳。
- 生活因素：运动、酒精、咖啡因、生病、补剂 / 健康项目。
- 成人健康：性生活、自慰、色情使用。
- 训练与目标：训练目标、签到、训练历史。
- 关系上下文：伴侣、关系情境、女性性健康关怀相关记录。

成人健康和关系上下文只展示覆盖率和样本限制，不把缺失当作问题。

## 提示方式

提示应面向行动：

- “睡眠记录覆盖较好，可以用于恢复相关观察。”
- “酒精记录较少，相关结论需要谨慎。”
- “某字段长期为空，可以考虑从首页弱化。”

## 不做

- 不自动删除字段。
- 不评价用户记录习惯好坏。
- 不要求补全历史。
- 不推断健康风险。
- 不把成人行为缺失当作问题。
- 不把低覆盖率字段从导出、备份或历史记录中移除。
