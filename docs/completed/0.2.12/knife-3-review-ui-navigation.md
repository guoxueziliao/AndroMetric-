# 0.2.12 刀 3 回顾 UI 与导航

> 本文定义阶段回顾的入口、页面结构和空状态。

## 入口

优先入口：

- Stats / 个人常态附近。
- 历史 / 复盘区域。
- Dashboard 仅可轻提示。

不新增顶级导航。

## 页面结构

建议结构：

- 窗口选择：本月 / 上月，可选本季度。
- 本阶段摘要。
- 分章节回顾。
- 记录缺口。
- limitations。

章节可以折叠或分区，但不要做报告式封面。

## 空状态

空状态应表达：

- “本阶段记录较少。”
- “这些章节暂时没有足够数据。”
- “部分来源尚未启用。”

空状态不表达：

- 用户失败。
- 数据不合格。
- 健康无法评估。
- 需要补打卡。

## Dashboard 边界

Dashboard 只展示：

- 是否有本月回顾可看。
- 简短入口文案。

Dashboard 不展示：

- 敏感指标值。
- 偏离方向。
- 伴侣或色情使用细节。

## 验收

- 用户能进入本月阶段回顾。
- 章节之间层级清晰。
- 移动端不出现长报告阅读压力。
- 没有分享、导出或打印入口。
