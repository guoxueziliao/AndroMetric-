# 0.2.12 刀 2 阶段回顾构建器

> 本文定义月度 / 季度阶段回顾如何由真实来源生成。

## 输入

- 回顾窗口：month 或 quarter。
- 起止日期。
- 刀 1 确认的可读来源。
- 每个来源的 sampleSize / confidence / limitations。
- 记录缺口。

## 构建顺序

1. 计算回顾窗口。
2. 拉取窗口内事实来源。
3. 按章节聚合。
4. 过滤未落地或无数据章节。
5. 附加 limitations。

## 章节边界

允许章节：

- 本阶段摘要。
- 个人常态变化。
- 值得回看的上下文。
- 观察计划。
- 经验卡。
- 记录缺口。

不允许章节：

- 健康评分。
- 阶段评级。
- 成功 / 失败。
- 风险等级。
- 未来预测。

## 输出

输出只服务 UI：

- `StageReview`
- `StageReviewSection[]`
- `limitations[]`
- `hiddenSections[]`

不写入 IndexedDB，不进入 JSON backup，不默认进入 CSV。

## 验收

- 缺数据时隐藏章节或显示空状态。
- 所有章节都能说明来源。
- 所有结论都是事实回看，不是诊断或因果。
