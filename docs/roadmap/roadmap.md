# Roadmap

> 总路线图只维护 0.2.4 及之后的当前规划状态。0.2.4 以前的完成版本默认不进入本轮规划维护。

## Status Legend

- `active`：当前开发线。
- `completed`：实现完成，已归档到 `completed/`。
- `planned`：已定方向，待执行。
- `discussion`：前移为下一轮讨论，但尚未定范围。
- `not doing`：长期明确不做。

## Planned

- 暂无 active / planned 版本。
- 0.2.21 已完成；0.2.22 已完成。
- 0.2.4 / 0.2.5 后发现的非数据安全反馈已由后续版本吸收或清理，不再维护独立反馈池。
