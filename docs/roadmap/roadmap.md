# Roadmap

## Status Legend

- `active`：当前开发线。
- `completed`：实现完成，已归档到 `completed/`。
- `planned`：已定方向，待执行。
- `discussion`：前移为下一轮讨论，但尚未定范围。
- `not doing`：长期明确不做。
